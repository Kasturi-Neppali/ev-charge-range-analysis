About Project

This project analyzes Electric Vehicle Charge and Range data using Python, SQL, Tableau and Flask. The dashboard provides insights into charging infrastructure, vehicle performance, pricing, efficiency, body styles and powertrain analysis. Tableau is used for interactive visualizations while Flask provides a web interface to access the dashboard and story.
